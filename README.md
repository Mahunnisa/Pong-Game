# Pong Game (Python Turtle)

A simple 2D Pong game built using Python's Turtle module. This game is inspired by the classic arcade Pong.

---

## Features
- Two-player game
- Paddle movement (W/S and Up/Down keys)
- Ball movement with collision detection
- Score tracking system
- Smooth gameplay using Turtle graphics

---

## Technologies Used
- Python
- Turtle module
